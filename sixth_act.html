CHANGE FONT SIZE

  • Output

  • Source Code

    Open PDF file:CHANGE_FONT-SIZE.pdf

  • Code Explanation

    This application has the option to change the font size, either increasing or decreasing it. By clicking the (+) button, the text size can be increased by five points. The (-) button reduces the text size by 5 points. The text can be resized by utilizing the (.setTextSize()) function, which is dependent on the size of the value that is passed into the function.
